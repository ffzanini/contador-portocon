# Imagens dos Departamentos

Esta pasta contém as imagens/ilustrações dos 3 departamentos da Protocon.

## Departamentos

- `fiscal.webp` - Departamento Fiscal: Ilustração relacionada a impostos/finanças
- `pessoal.webp` - Departamento Pessoal: Ilustração relacionada a funcionários/equipe
- `contabil.webp` - Departamento Contábil: Ilustração relacionada a organização/documentos

## Formato Recomendado

- Formato: WebP (para melhor performance)
- Dimensões: 400x400px (aspect ratio 1:1)
- Otimização: Compressão adequada para web
